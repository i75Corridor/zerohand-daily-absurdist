# The Daily Absurdist

A Zerohand pipeline package for generating satirical news articles.

## Pipeline

`Research → Write → Edit → Illustrate → Publish`

Given a topic, the pipeline:
1. **Researches** the topic with web searches
2. **Writes** a satirical article (2,000–2,500 words, dry deadpan tone)
3. **Edits** for quality and extracts image prompt
4. **Illustrates** with AI image generation
5. **Publishes** the final article

## Usage

Install via the Zerohand Packages page by searching for this repo or using the URL:

```
https://github.com/i75Corridor/zerohand-daily-absurdist
```

## Skills Included

- `researcher` — web search and topic research
- `writer` — satirical article writing
- `editor` — article review and editing
- `imagen` — AI image generation
- `publisher` — article publishing

## Input

```json
{ "topic": "Your topic here" }
```
